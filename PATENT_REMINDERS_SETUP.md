# Напоминания по патентам

Модуль отправляет уведомления в Telegram **за 7 и 3 дня** до окончания патента.

## Логика

- Проверка каждый день в **09:00 МСК** (встроенный scheduler ERP).
- Напоминание `day7` — в окне 4–7 дней до окончания (с догоном, если день пропущен).
- Напоминание `day3` — в окне 0–3 дней до окончания (с догоном).
- Каждый тип напоминания по периоду отправляется **один раз**.
- Перед отправкой проверяется, есть ли оплата в окне напоминания — если да, уведомление не уходит.
- В тексте всегда фиксированная формулировка «напоминание за 7/3 дн.» (без ежедневного обратного отсчёта).

## Переменные окружения

Обязательно:

- `TG_BOT_TOKEN` — токен Telegram-бота

Рекомендуется:

- `TG_CHAT_ID_PATENTS` — отдельный чат для напоминаний по патентам

Если `TG_CHAT_ID_PATENTS` не задан, используется `TG_CHAT_ID_HR`, затем `TG_CHAT_ID`.

## Ручной запуск (диагностика)

Endpoint остаётся для ручной проверки:

- URL: `/patents/reminders/run`
- Методы: `GET` или `POST`
- Авторизация: `PATENT_REMINDER_TOKEN` в query или header `X-Reminder-Token`

```bash
curl "https://YOUR_DOMAIN/patents/reminders/run?token=YOUR_SECRET"
```

Внешний cron **не обязателен** — задача уже в `app/scheduler.py`.
